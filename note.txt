2. ONE-TIME SETUP
================================================================================

STEP 1: Get Your Admin Telegram ID
-----------------------------------
1. Open Telegram ‚Üí Search "@userinfobot"
2. Start chat ‚Üí It shows your ID (e.g., 123456789)
3. SAVE this number!

STEP 2: Install Dependencies
-----------------------------
Open Terminal/Command Prompt in the SOCHOW folder:

Mac:
  cd /path/to/sochow
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt

Windows:
  cd C:\path\to\sochow
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt

STEP 3: Create .env Configuration File
---------------------------------------
In the SOCHOW folder, create a file named ".env" (include the dot!)

Paste this and fill in YOUR details:

BOT_TOKEN=your_bot_token_here
ADMIN_CHAT_ID=your_telegram_id_from_step_1
PAYMENT_BANK=First Bank
PAYMENT_ACCOUNT=1234567890
PAYMENT_NAME=SOCHOW
PORT=3000

Replace:
- BOT_TOKEN: Get from developer who created @sochow_food_bot
- ADMIN_CHAT_ID: Your ID from Step 1
- PAYMENT_BANK, PAYMENT_ACCOUNT, PAYMENT_NAME: Your payment details

Save and close.
